shinyroom is a Shiny app for image manipulation.

Work in progress.

## Getting Started

```sh
git clone git@github.com:igjit/shinyroom.git
cd shinyroom
docker build -t shinyroom .
docker run -it --rm -p 8000:8000 -e PORT=8000 shinyroom
```

and open http://localhost:8000/
